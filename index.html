Git Commands

  • git init - Initialize a new Git repository
  • git clone [repository_url] - Clone a remote repository
  • git add [file] - Add a file to the staging area
  • git commit -m "[message]" - Commit changes with a message
  • git push - Push changes to a remote repository
  • git pull - Pull latest changes from a remote repository
  • git branch - List all branches in the repository
  • git checkout [branch_name] - Switch to a different branch
  • git merge [branch_name] - Merge changes from a branch
  • git status - Show the current status of the repository
